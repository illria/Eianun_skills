# Dialogue Translation Reference

Use this reference when a story contains Chinese dialogue that should become English spoken dialogue in the final prompt.

## Default Rule

- Storyboard table: keep Chinese original and add English translation.
- Final prompt: use English only inside spoken braces `{}`.
- Do not include Chinese and English versions of the same line in the final prompt unless the user explicitly asks.

## Style Rules

Translate for performance, not literal subtitles.

- Fear: broken, breathy, fragmented.
- Ancient/fantasy tragedy: slightly elevated but natural.
- Sarcasm: concise and cutting.
- Tenderness: simple and intimate.
- Threat: controlled, not over-explained.

## Examples

| Chinese | English spoken line |
|---|---|
| 不要……放开我……我不想嫁给海神…… | No... let me go... I don't want to marry the sea god... |
| 你……一夜没睡吗？ | You... stayed awake all night? |
| 我习惯了。 | I'm used to it. |
| 你不怕我趁你睡着的时候逃跑？ | Aren't you afraid I'll run away while you sleep? |
| 不怕。 | No. |
| 因为外面的人，比我可怕得多。 | Because the people outside are far more terrifying than I am. |
| 至少你没有假装是为我好。 | At least you never pretended it was for my own good. |
| 吃。蛇在海里抓的。 | Eat. The snakes caught them in the sea. |
| 比绝大多数人可靠。 | More reliable than most people. |
| 我是神庙最纯洁的祭司。 | I was the purest priestess in the temple. |
