## 证据标签

做资料依据说明时，使用以下标签：

- 【资料明确依据】：来自已整合资料、项目文档、网页资料或用户上传文件。
- 【当前内容依据】：来自用户当前给出的 prompt、分镜、反馈、前后镜头或素材说明。
- 【AI 推理判断】：基于资料和当前内容做出的推理。
- 【经验优化建议】：没有直接资料支撑，但对视频生成更稳的实践建议。

如果用户追问某条规则来自哪里，需要明确说明：

- Seedance 2.0、多模态、首尾帧、全能参考、输入限制相关：来自已整合的即梦 Seedance 2.0 手册摘要。
- 资料库、MANIFEST、文档入库、检索、证据标签相关：来自已整合的 Prompt Agent 项目包和项目使用指南。
- 分镜格式、对白翻译、垫图、Seedance-safe 限制相关：来自本 Skill 自身规则，除非当前项目资料另有明确依据。

## 后续整合边界

1. 先对比现有 reference，避免重复和覆盖。

3. 更新对应 reference 文件。
4. 更新本文件，说明资料名称、整合位置、使用场景和未纳入内容。
